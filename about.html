# Rony Bolivar Art Website A complete static website for your art portfolio, blog, and online shop. ## Files Included - index.html (Homepage) - gallery.html (Shop) - blog.html - about.html - contact.html - styles.css - script.js - images/ (add your artwork images here) ## Setup 1. Replace placeholder images in `images/` folder with your actual JPG/PNG files (rename to match references or update paths in HTML/JS). 2. Customize text, prices, artworks in `script.js` and HTML files. 3. Open `index.html` in a browser to test. 4. For live site: Upload to GitHub and enable GitHub Pages, or use Netlify/Vercel (drag & drop folder). ## Customization Tips - Add more artworks to the `artworks` array in script.js. - For real payments: Integrate PayPal button or Stripe. - SEO: Add meta tags, alt texts. - Blog: Expand with more posts or use a static site generator later. Contact for further customizations! Enjoy your new art site. 🎨 © 2026 Rony Bolivar
